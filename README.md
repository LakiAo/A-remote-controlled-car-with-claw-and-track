# A-remote-controlled-car-with-claw-and-track
一个带爪子和循迹的小车，用于学校机器人矿塔争夺赛，  
基于[**@ptsfdtz**](https://github.com/ptsfdtz)的代码进行编写  
队友[**@O2HCHO**](https://github.com/O2HCHO)  

使用麦轮+四路循迹+单自由度的爪子完成学校矿塔争夺赛  
使用PS2手柄控制
